# Pro Task Manager

![Task Manager Screenshot](screenshot.png)

## Description
A modern, advanced task management web application built with **HTML, CSS, and Vanilla JavaScript**.  
Users can add, edit, delete, set reminders, reorder tasks with drag & drop, search, filter, and track completed/overdue tasks. Data is persisted in **localStorage** for seamless experience.

## Features
- Add / Edit / Delete tasks
- Set date & time reminders
- Browser notifications for reminders
- Drag & Drop task reorder
- Search tasks by keyword
- Filter tasks: All / Completed / Pending / Overdue
- Dark / Light mode toggle
- Task statistics counter
- Overdue task highlight
- Persistent data with localStorage

## Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
- **localStorage**
